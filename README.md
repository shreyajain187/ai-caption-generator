## AI Caption Generator

This project generates captions for uploaded videos using OpenAI Whisper.

## Features
- Upload video
- Generate captions automatically
- Display subtitles on video
- Download subtitles

## Tech Stack
- Python
- Flask
- Whisper
- FFmpeg

## How to Run
1. Install dependencies
2. Run app.py
3. Open http://127.0.0.1:5000
